# Road-Traffic-Accident-Severity-Prediction


## Overview
This project predicts road accident severity using machine learning techniques.

## Dataset Features
- Weather
- Road Type
- Time of Day
- Traffic Density
- Speed Limit
- Driver Age
- Driver Experience
- Vehicle Type
- Road Condition

## Models Used
- Decision Tree
- Random Forest
- XGBoost

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Technologies
- Python
- Pandas
- Scikit-Learn
- XGBoost
- Matplotlib
- Seaborn

## Results
Random Forest and XGBoost achieved strong performance for accident severity classification.
